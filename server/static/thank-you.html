Windows SmartScreen Warning?

Windows may show a "Windows protected your PC" warning because Hang is new and building reputation. This is normal for open-source apps.

How to install anyway
  1. Click "More info" on the warning popup
  2. Click "Run anyway" button
  3. The installer will start normally
